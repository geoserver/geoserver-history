Mapbuilder demo pages

Simple Map Viewer
A minimal map viewer which allows a user to view layers from Web Map Services (WMS) or load a Web Map Context (WMC) document which describes layers to view. This is a good demo to copy when building a new application.
Status: beta
Requirements:
  • Browsers supported are Internet Explorer 6 or later, Mozilla 1.3 or later, Netscape 6 or later.
WMS Time Series
Shows maps as a time series. Zoom around.
Status: alpha
Requirements:
  • Browsers supported are Internet Explorer 6 or later, Mozilla 1.3 or later, Netscape 6 or later.
Complete configuration
Not pretty to look at but shows integration of most components. Intended for testing extensions to the code base.
Status: alpha
Requirements:
  • Some functionality requires Mapbuilder to be installed inside a Java Servlet Container (like Tomcat).
  • Browsers supported are Internet Explorer 6 or later, Mozilla 1.3 or later, Netscape 6 or later.
Feature Entry
Shows editing and updating of GML features and uploading to a Transactional Web Feature Service (WFS-T).
Status: prototype
Requirements:
  • Mapbuilder needs to be installed in a Java Servlet container in order to upload events to a WFS-T
  • This demo accesses data from a default Geoserver installation which should be installed in the same Java Servlet Webapps directory as Mapbuilder.
  • Browsers supported are Internet Explorer 6 or later, Mozilla 1.3 or later, Netscape 6 or later.
Context Editor
