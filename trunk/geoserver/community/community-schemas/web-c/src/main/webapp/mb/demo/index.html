Mapbuilder Demo Pages

Simple Map Viewer
A minimal map implementation showing a map embedded in a page.
Basic Map Viewer
A basic map viewer implementation. This is a good demo to copy when building a new application.
Demis Maps Viewer
A Demis maps viewer which allows a user to switch maps Google style.
WMS Time Series
Shows maps as a time series. Zoom in to a region of interest.
Feature Entry
Shows editing and updating of GML features and uploading to a Transactional Web Feature Service (WFS-T). This demo accesses data from a Geoserver installation installed in the same Java Servlet Webapps directory as Mapbuilder.
Context Editor
Creation and editing of Web Map Context documents from searching and browsing a registry.
  • Browsers supported are Internet Explorer 6 or later, Mozilla 1.3 or later, Netscape 6 or later.
  • Some functionality requires the server proxy and serialize scripts in the configuration file. Implementations of these are available for PHP enabled servers and Java Servlet Container (like Tomcat).
